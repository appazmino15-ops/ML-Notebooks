\# Neurona desde cero



Proyecto de aprendizaje para implementar y entender una neurona artificial

desde cero usando Python y Jupyter Notebook.



\## Objetivo

\- Entender cómo funciona una neurona artificial

\- Implementar el modelo paso a paso

\- Entrenar y evaluar con datos simples para un sistema de alarmas



\## Tecnologías

\- Python

\- NumPy

\- Jupyter Notebook
